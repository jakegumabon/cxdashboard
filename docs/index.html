What is Adjusted Volume?
The CX data tracks — new tickets created in the selected period. But agents also handle reopened tickets — cases that were previously solved but came back. Operational Volume data shows — reopens, bringing the true total to — tickets.

Since reopens can't be tracked per agent individually, we apply the ops ratio (—) uniformly — each agent's workload is estimated at — their new ticket count. This is an approximation; actual distribution may vary by agent.
Tickets / Day and Tickets / Hour
Tickets / Day = Adjusted Volume ÷ Work Days in the selected period.
Tickets / Hour = Adjusted Volume ÷ Total Work Hours in the selected period.

These show the estimated ticket load per agent relative to their actual schedule. Higher numbers mean heavier workload per shift, making it easier to spot imbalances across agents in the same team.

How to Interpret the Charts
The Agent Throughput chart ranks agents by tickets per work day. A longer bar means more volume relative to scheduled hours — useful for spotting who's carrying the most load.

The Intraday Load chart shows which hours are busiest per team. This helps with planning coverage and identifying gaps in shift schedules.

Day × Hour Matrix — Avg Tickets per Slot
Shaded cells = agent scheduled. Bright = more tickets. Dim = off-shift (tickets may still arrive but no agent is rostered).
Each cell = how many tickets one agent would need to handle in that 1-hour slot, on average.

Here's the formula: avg arrivals × ops ratio ÷ agents scheduled

We start with the average ticket arrivals for that day/hour combination across the full loaded period (—). We then multiply by the ops ratio (—) — this accounts for reopened tickets, since agents handle both new and reopened cases and not just the newly created ones. Finally, we divide by the number of agents scheduled in that slot based on their individual shift rosters.

Color intensity = workload level. Brighter/more saturated = more tickets per agent per hour. ■ Dark gray = no agents rostered (tickets still arrive but nobody is assigned to handle them).
